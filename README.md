# HR Analytics & Employee Attrition Prediction Dashboard

## Project Overview
This project focuses on analyzing employee attrition trends and workforce insights using Power BI and Excel. The dashboard helps identify key factors affecting employee retention such as salary, overtime, work-life balance, department, and job role.

The objective of this project is to support HR decision-making through data-driven insights and interactive visualizations.

---

## Tools & Technologies
- Power BI
- Microsoft Excel
- SQL
- DAX
- Data Visualization

---

## Key Features
- Employee Attrition Analysis
- Attrition Risk Classification
- Department-wise Attrition Insights
- Salary Analysis
- Overtime Analysis
- Work-Life Balance Analysis
- KPI Cards & HR Metrics
- Interactive Dashboard Filters
